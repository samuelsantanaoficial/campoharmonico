Gerador de campo harmônico

O campo harmônico é um conjunto de acordes formados a partir de uma escala específica, seguindo uma estrutura lógica baseada em terças. Ele define quais acordes soam naturalmente bem juntos dentro de uma tonalidade, servindo como base para composições, arranjos e improvisações. Cada campo harmônico pode ser maior, menor natural, menor harmônico ou menor melódico, e entender sua formação é essencial para criar progressões harmônicas equilibradas e musicais.
